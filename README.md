# Proyecto-Final
Proyecto Final "Astro Guide: PocketSolar

El proposito de Astro Guide: PocketSolar

El proposito de mi aplicacion Astro Guide: PocketSolar es brindarle informacion directa de lo que exactamente deseen. Ademas la pagina web les ofrecer a cada uno de los usuarios una experiencia satisfactoria y facil en lo que es la busqueda de información del planeta que deseen buscar.

Tecnologia Utilizada

La tecnologia utilizadas son: HTML, CSS y JavaScript. 

Como instalar y ejecutar el sitio

Para ejecutar y instalar usted debe descomprimir el zip file y entra a visual studio code. Cuando entre a visual studio code debe pasar a la parte en la cual le dice "Run and Debug" cuando este en esa parte le va a dar click encima de el texto. Cuando pase el rato usted debe de estar dentro de la pagina web Astro Guide: PocketSolar.

Informacion relevante 

Algunas de las funcionalidades de Astro Guide: PocketSolar Las Funcionalidades mi app es obtener información diversas de diferentes tipos de planetas. En la cual esta información se puede diversificar con los cambios q pueden pasar en el dia a dia.

En esta pagina web puede encontrar variedad de planetas en la cual contiene factos y informacion de cada una de ellas. Esta informacion son sacadas de una fuente confiable en la cual son contraidas directamente de de noticias diarias de cada uno de los planetas.

Reflexion 

Esta pagina web es dedicada para cada una de esas personas curiosas y quisieran saber mas de lo que esta en la galaxia. Mi experiencia creando esta pagina fue agradable ya que es algo q me gusta buscar y saber mas cada dia.
